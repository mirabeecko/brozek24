Jak získat Stripe Payment Link
  1. Přihlaste se na dashboard.stripe.com
  2. Vlevo menu → Payment Links → tlačítko + New
  3. Vytvořte produkt (název, cena v CZK), klikněte Create link
  4. Zkopírujte URL (začíná https://buy.stripe.com/…) a vložte do pole níže
  5. Opakujte pro každou službu zvlášť
